# Password-manager-on-XOR

This is Password manager on XOR, that uses flash drive to keep user passwords.

**How to use GUI**
1) Create master-password (must be at least 8 characters long and contain numbers, capital letters and special characters).
2) Enter using new master-password
- **To save password:**
    In the white field, enter the password you want to save and press "Save".
- **To delete password:**
     Press red button "Delete" next to the password.
- **To auto-type password:**
    Press "Select" button next to the password, then press "p" on keyboard in the field where you want to enter the password.
- **To move data:**
    Press "Move" button, then connect the external device to the computer you want to start working on, launch the password manager and click "Restore" buuton.
